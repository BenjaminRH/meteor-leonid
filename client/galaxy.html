Commands

  • !mrt / !meteorite: Basic information about Meteorite
  • !atmo / !atmosphere: Basic information about Atmosphere
  • !package : Lookup an Atmosphere package
  • !np: Provides a "no problem" message when a user thanks you for your help
  • !dataja: "Don't ask to ask, Just ask!"
  • !ugt: Basic information about Universal Greeting Time
